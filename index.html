1 - Présentation et explication (1/2)

  • Tests "end to end"
  • Uniquement basés sur le FRONT
  • Utilisés pour vérifier l'intégrité du fonctionnement FRONT
  • Se positionnent comme un outil de prévention
  • Ne doivent pas remplacer les tests fonctionnels
  • Se basent sur le driver JS de Selenium pour fonctionner
  • Sont complémentaires des autres types de tests
  • Devraient être réalisés quand l'application est en RC

2 - Pré-requis

  • Avoir une version de l'application en RC (ou équivalant)
  • Très bonne connaissance fonctionnelle de l'application
  • Bonne connaissance du Javascript
  • Utilisation de NodeJS et de NPM
  • Pour Protractor, avoir une application AngularJS

3 - Installation des pré-requis (1/4)

  • Installation de NodeJS
  • Utilisation du gestionnaire dépendances NPM
  • Téléchargement des drivers Selenimum et Chrome pour Protractor et NightWatch
    • Téléchargement du driver pour le server Selenium
    • Téléchargement du driver pour intégrer Chrome
    • Déjà fournis dans le repository
      ./bin/

3 - Installation de Protractor (2/4)

  • Installation de Protractor pour du AngularJS 
    • npm install -g protractor
  • Mise en place de la configuration avec les drivers
  • module.exports.config = {
  framework: 'jasmine',
  specs: ['protractor/tests/**/*.js'],
  seleniumServerJar: './bin/selenium-server-standalone-2.52.0.jar',
  chromeDriver: './bin/chromedriver',
  capabilities: {
    'browserName': 'chrome'
  },
}

3 - Installation de NightWatch (3/4)

  • Installation de NightWatch 
    • npm install -g nightwatch
  • Mise en place de la configuration avec les drivers
  • {
  "src_folders" : ["nightwatch/tests"],
  "output_folder" : "nightwatch-reports",
  "selenium" : {
    "start_process" : true,
    "server_path" : "./bin/selenium-server-standalone-2.52.0.jar",
    "cli_args" : {
      "webdriver.chrome.driver" : "./bin/chromedriver",
    }
  }
}

3 - Installation de CasperJS (4/4)

  • Installation de CasperJS 
    • npm install -g casperjs
  • Pas de drivers
  • Utilisation de PhantomJS pour WebKit 
    • npm install -g phantomjs
  • Utilisation de SlimmerJS pour firefox 
    • npm install -g slimmerjs
